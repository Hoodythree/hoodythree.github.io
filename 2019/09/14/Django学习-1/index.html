初识Django

发表于 2019-09-14 更新于 2019-09-17

1.Django是什么?

Django是基于python的Web Application Framework,引用官网的一段介绍:

Django is a high-level Python Web framework that encourages rapid development and clean, pragmatic design. Built by experienced developers, it takes care of much of the hassle of Web development, so you can focus on writing your app without needing to reinvent the wheel. It’s free and open source.

2.为什么使用Django?

对我而言,我需要一个快速开发框架,最好能包括尽量多的工具,Django完美满足这些特点.
与Flask相比,Django提供了大量的工具,比如:

  1. 表单–Django Form
  2. csrftoken登陆认证鉴权–Django 自带了 backend 和 auth 模块。
  3. ORM–Django ORM
  4. DBMigration–Django Migration
  5. 缓存–Django Cache…
    做到了开箱即用.

3.如何使用Django?

  • 在Windows上安装Django

    step1. 安装python和pip包管理工具
    step2. 安装virtualenv和virtualenvwrapper这两个工具

    virtualenv:用于创建一个独立的虚拟python环境,其中每个环境都有自己的库和site-package
vittualenvwrapper:简化virtualenv的使用

    step3. Install Django 

    pip install django

  • Pycharm搭建第一个Diango项目
    有两种方法创建一个Django应用,一是在创建Django项目的同时创建一个Django应用,二是通过在Terminal中输入以下命令:

    py manage.py startapp appname

    Django应用的目录结构如下:

    +yourApp/
    + migrations/
        __init__.py
    __init__.py
    admin.py
    apps.py
    models.py
    tests.py
    views.py

    其中各个文件的作用如下:

    • migration :用于连接网站和数据库
    • __initial.py:告诉python将这个app视作一个python应用
    • admin.py:每个网站都会有一个admin section,用于实现管理功能
    • apps.py:app的配置文件
    • models.py: 我的理解是model是数据库的一个映射,它指明了app中的数据是如何进入到数据库的
    • test.py:测试文件,用于debug
    • views.py:views本质上是python函数,接受用户的request并respond相应的结果

  • Hello world in Django
    首先,在yourApp/views下创建一个view

    from django.http import HttpResponse

def index(request):
return HttpResponse('<h1>Hello World</h1>')

    然后,在yourApp下创建一个urls.py文件,定义一个url pattern

    from django.urls import path
from . import views

urlpatterns = [
    path('', views.index, name='index')
]

    最后,将yourApp/urls.py中创建的url pattern加入到根目录下urls.py的path中

    from django.contrib import admin
from django.urls import path, include

urlpatterns = [
    path('admin/', admin.site.urls),
    path('helloworld/', include('HelloWorld.urls')
]

  • 运行Hello World
    在浏览器中输入http://127.0.0.1:8000/helloworld/
    Got it!
